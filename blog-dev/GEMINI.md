
# dev.svachmic.cz

This is a technical blog where the author, Michal Švácha, writes about software development, programming, and related topics.

## Running Locally

1.  Navigate to the `blog-dev` directory:
    ```bash
    cd blog-dev
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run develop
    ```
4.  Open your browser and navigate to `http://localhost:8000`.
