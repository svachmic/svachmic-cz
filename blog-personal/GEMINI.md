# blog.svachmic.cz

This is a personal blog for Michal Švácha's thoughts and writings on various topics.

## Running Locally

1.  Navigate to the `blog-personal` directory:
    ```bash
    cd blog-personal
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run develop
    ```
4.  Open your browser and navigate to `http://localhost:8000`.
